## Instalation

Clone this repository to your local environment:

```
git clone https://github.com/Luciano-Holz/cpf-validation.git
cd cpf-validation 
```

```
npm install
```

## Requirements
[Node.js](https://nodejs.org/en/)

##  Using

To run locally

Run as developer:

```
npm run dev
```
